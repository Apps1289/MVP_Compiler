# MVP_Compiler

A minimal MVP compiler demo that visualizes source code as:
- token stream
- abstract syntax tree (AST)

## Run
Open `/home/runner/work/MVP_Compiler/MVP_Compiler/index.html` in a browser.

## Supported syntax
- `let name = expression;`
- `print expression;`
- expression statements ending in `;`

Expressions support identifiers, numbers, parentheses, and `+ - * /`.
